THE ULTIMATE SIMULATOR 3000

Description: This project revolves around the simulation of natural habitats. Users can choose a habitat,
a climate change scenario, as well as a number of various animals. It simulates the animals' interactions
in the selected environment. Simulations can be ended to start a new one and users can interact with their
simulation in different ways through choosing the number of steps they want to run.

!IMPORTANT! Launching the project:
The project's main class is Initializer and not Simulator. To start a new session,
users must create an instance of Initializer NOT Simulator.
